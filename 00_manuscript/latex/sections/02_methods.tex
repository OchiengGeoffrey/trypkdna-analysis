\subsubsection{Genome Assembly and Quality}

The \teq{} OVI genome (RefSeq GCA\_001457755.2) was assessed for completeness using BUSCO v6.0.0 with the \texttt{trypanosoma\_odb12} lineage (270 conserved genes).

\subsubsection{BLAST Database and ATPase γ Detection}

A nucleotide BLAST database was constructed from the \teq{} genome. The \tbr{} ATPase γ protein (UniProt Q57XH6) was queried using \texttt{tblastn} with e-value threshold 1e-10.

\subsubsection{SRA Toolkit Validation}

The accession SRR390436 was downloaded using \texttt{prefetch} and converted to FASTQ using \texttt{fasterq-dump}.
